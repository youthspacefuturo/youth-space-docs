---

## Como adicionar novos recursos

1. Criar pasta dentro de `recursos/[tema]/[nome-do-material]/`
2. Criar arquivo `info.md` como entrada da pasta
3. Adicionar links no módulo relevante

**Exemplo de estrutura:**
```
recursos/
├── dax/
│   └── vendas-mais/
├── python/
│   └── pandas-exercises/
└── javascript/
    └── async-exercises/
```
